# README

## Requirements

This project requires Java 17 SDK or newer (tested with temurin-17.0.5).

## Compilation

- Navigate to the "src" directory
- Run `javac SAIS.java`

## Execution

- Navigate to the "src" directory
- Run `java SAIS path_to_input_file` whereas "path_to_input_file" stands for the path to the input file
